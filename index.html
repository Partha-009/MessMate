⚙️ One-Time Firebase Setup Required

MessMate needs a free Firebase database so all your roommates share the same data in real time. Follow these steps once and you're done forever!

  1. Go to console.firebase.google.com and sign in with Google
  2. Click Add project → name it MessMate → Continue → Create project
  3. Click the </> (Web) icon → name it messmate → click Register app
  4. You'll see a firebaseConfig block — copy all 7 values from it
  5. Open this HTML file in a text editor → find the firebaseConfig section at the bottom → replace each PASTE_YOUR_..._HERE with your actual values
  6. In Firebase Console → left menu → Authentication → Get started → Enable Email/Password → Save
  7. In Firebase Console → left menu → Realtime Database → Create database → choose your region → Start in test mode → Enable
  8. Go to Realtime Database → Rules tab → replace all content with:
    {"rules":{".read":"auth != null",".write":"auth != null"}} → Publish
  9. Upload the updated file to GitHub and share your link! 🎉
⚠️ The databaseURL must look like: https://your-project-default-rtdb.firebaseio.com (or with a region like .asia-southeast1.firebasedatabase.app)
